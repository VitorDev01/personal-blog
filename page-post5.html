Oque é o protocolo FTP

A sigla FTP File transfer protocol é um protocolo de rede para a transmissão de arquivos entre computadores. Dentro do conjunto TCP/IP (Transmission Control Protocol/Internet Protocol), ele é um protocolo da camada de aplicação para o download e upload de arquivos em conexões do tipo cliente/servidor.

Como usar

Primeiro voce precisará estar conectado a mesma rede wi-fi nos dois dispositivos um que ira fazer a transferência para o que ira receber.

De Celular Para Celular : No sistema operacional android o recurso do protocolo ftp vem desativado na maioria dos aparelhos e para ativar basta baixar na playstore o aplicativo WI-FI FTP SERVER,ao abrir o aplicativo aparecerá o botão de ligar e desligar.

Ao ligar ele ira gerar um link com o ip do dispositivo, a tempos atrás era possivel acessar os arquivos através desse link no navegador mas atualmente essa opção foi bloqueada,e como alternativa para isso usaremos o aplicativo Es File Explore que ja vem com o recurso ftp disponivel para uso logo abaixo vou o link para baixar o app e um tutorial de como usar o ftp nele.
